# 🏥 SmartCare Hospital Management System

SmartCare Hospital Management System is a Java-based desktop application designed to simplify and manage hospital operations efficiently. The system provides a user-friendly graphical interface developed 
using Java Swing and integrates with a MySQL database through JDBC for secure data management.

This application allows hospital staff to perform essential administrative tasks such as patient registration, room allocation, employee management, ambulance tracking, patient discharge, and viewing patient 
records. By digitizing these processes, the system helps reduce manual work, improve data organization, and enhance overall hospital management efficiency.

The project demonstrates the implementation of database connectivity, GUI development, and modular programming using Java, making it a useful educational project for understanding real-world healthcare 
management systems.

# 🛠️ Technologies Used

- **Programming Language:** Java  
- **GUI Framework:** Java Swing  
- **Database:** MySQL  
- **Database Connectivity:** JDBC  
- **IDE:** NetBeans / Eclipse / IntelliJ IDEA
